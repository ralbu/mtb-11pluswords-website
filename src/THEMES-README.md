# 11+ Words Website - Color Themes

This project includes **3 different color theme versions** of the website, each with its own unique color palette.

## Theme Files

### Theme 1: Coral & Blue (Warm & Energetic)
- **Files**: `index-theme1.html` + `styles-theme1.css`
- **Color Palette**:
  - Primary: Coral/Peach (#F3666B, #F68C90, #FBBEA0)
  - Secondary: Blue & Green (#3493F1, #13A042)
  - Text: Deep Red (#420002, #BA5B5B)
  - Background: Light Pink/Grey (#FFFAFA, #F4F4F4)
- **Mood**: Warm, friendly, energetic

### Theme 2: Ocean Blue & Teal (Professional & Trustworthy)
- **Files**: `index-theme2.html` + `styles-theme2.css`
- **Color Palette**:
  - Primary: Ocean Blue (#0096c7, #00b4d8)
  - Secondary: Deep Blue & Teal (#023e8a, #0077b6)
  - Text: Navy (#0a2540, #1e5a7d)
  - Background: Light Blue (#e6f7ff, #d9f0f7)
- **Mood**: Professional, trustworthy, calm

### Theme 3: Forest Green & Gold (Natural & Fresh)
- **Files**: `index-theme3.html` + `styles-theme3.css`
- **Color Palette**:
  - Primary: Forest Green (#40916c, #52b788)
  - Secondary: Dark Green (#1b4332, #2d6a4f)
  - Text: Deep Forest (#1b4332, #2d6a4f)
  - Background: Soft Green/Cream (#f1faee, #e8f5e3)
- **Mood**: Natural, fresh, calming

## Original Version
- **Files**: `index.html` + `styles.css`
- **Color Palette**: Purple gradient (original design)

## How to View

Simply open any of the HTML files in your browser:
- `index-theme1.html` - Coral & Blue theme
- `index-theme2.html` - Ocean Blue theme
- `index-theme3.html` - Forest Green theme
- `index.html` - Original purple theme

## Features

All themes include:
- Clean semantic HTML
- Mobile-first responsive design
- Modern styling with gradients and shadows
- CSS-only phone mockup
- Responsive grid layouts
- Abstract background pattern in footer
- Smooth hover animations
- SEO metadata

## Theme Selection Guide

Choose a theme based on your brand identity:

- **Theme 1 (Coral)**: Best for a friendly, approachable, energetic brand
- **Theme 2 (Blue)**: Best for a professional, trustworthy, educational brand
- **Theme 3 (Green)**: Best for a natural, calm, growth-oriented brand
